# 🔐 Cadence

A simple Streamlit application that detects suspicious login attempts using user behaviour.

## Features

- Checks login time
- Checks device
- Checks location
- Checks impossible travel
- Calculates a risk score
- Gives one of three decisions:
  - ✅ Allow
  - ⚠️ Challenge
  - ❌ Block

## Technologies Used

- Python
- Streamlit

## Run the Project

Install the required package:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
streamlit run app.py
```
